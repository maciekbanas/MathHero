# Changelog

## development version

Initial release of `MathHero` game.

### Added

* Screens:
  * Choose character.
  * Select land.
  * Math battle.
  * Inventory.
* Characters:
  * Knight.
  * Sorceress.
  * Wolf-warrior.
  * Mage.
* Maps/realms:
  * Goblin forest.
  * Steel hills.
  * Mushroom swamps.
  * Ice realm.
* Enemies:
  * Goblins.
  * Gnomes.
  * Trolls.
  * Mushroomkins.
  * Golems.
  * Wolves.
* Items:
  * Berries.
* Game Logic:
  * Moving on square grid.
  * Fighting with enemies as finding solution to mathematic equations.
  * Adding life bar to heroes.
  * Differentiating equations with regard to different enemies.
  * Gathering coins after winning over enemies.
